This was a group Cisco Packet Tracer project for my Network Security module, where we designed and configured a simulated enterprise network for BrightSoft Solutions across three floors. Our work included subnetting, DHCP, DNS, VLAN segmentation, inter-VLAN routing, ACLs, NAT, and secure external access to a staging server. We also implemented routing using RIP and OSPF, together with failover considerations to improve resilience and monitoring. The final solution was completed and documented collaboratively by Harun Ulvi Denli, Noor Dyer, and Lesetja Matsena, with all three members credited on the submission coversheet.

To view the simulated network:

1. Download Cisco Packet Tracer and Sign-Up
2. Download and open the 'Project File.pkt'
3. Wait for the network to load and test each requirement
